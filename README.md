**Интернет-браузер**

   Приложение демонстрирует мои навыки работы с фреймворком WebKit, паттерном Observer и нестрандартной цветовой палитрой (Hex-Color).

При разработке использовались:
* WebKit
* UIScrollView
* Hex-Color
* Работа с расширениями
